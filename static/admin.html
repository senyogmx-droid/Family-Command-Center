Gateway DHCP Static Reservation Guide

To guarantee the kiosk screens, smart sockets, and remote tablet deep links never lose connectivity, configure a static DHCP IP reservation in your household router.

1.
Log into your Router Administration Gateway panel (typically http://192.168.1.1 or 10.0.0.1) using a web browser.
2.
Navigate to the LAN Settings / DHCP Server / Address Reservation settings tab area.
3.
Add a new Reservation record. Bind the MAC address of this host machine (Loading...) to the desired static IP (Loading...).
4.
Save settings, apply configurations, and reboot your router. Your Family Command Center is now locked securely in place.
